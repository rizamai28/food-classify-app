# 【このアプリの概要】 food-classify-app
食べ物の画像を認識して健康度を判定するアプリです。  
あなたが撮った写真や画像をアップロードすることで、AIがどれだけ健康的な画像であるかを判定します。  
レスポンシブに対応してるのでスマホからもご覧いただけます。  
<br>
